## 💎 A Experiência Elite

A **A10 Fitness Elite** não é apenas um fitness tracker; é um ecossistema de alta performance desenvolvido para atletas que buscam a precisão do **Padrão Square Elite**. Do rastreio de volume individual à competição em tempo real no Ranking Global, cada detalhe foi forjado para máxima eficiência e estética retro-moderna.

### ⚡ Destaques da Plataforma

- **🛡️ Protocolo de Confiabilidade**: Evolua sua badge de Bronze até o lendário status **Diamante** com base na consistência e validação de seus treinos.
- **🔥 Forja Social**: Feed interativo em tempo real onde você pode apoiar companheiros de treino, comentar estratégias e seguir os maiores nomes da academia.
- **🏆 Gamificação Progressiva**: Sistema de XP multinível (Iniciante a Lenda), badges exclusivas e streaks ardentes que transformam cada gota de suor em progresso visível.
- **📊 Analytics de Alta Precisão**: Gráficos de tendência de volume (7 dias e mensais), recordes pessoais (1RM) automatizados e métricas de biometria.
- **📱 PWA de Próxima Geração**: Experiência mobile nativa fluida, com suporte completo a Progressive Web App para treinar em qualquer lugar, sem atritos.

## 🚀 Início Rápido (Local)

1. **Clonar Repositório**:
   ```bash
   git clone https://github.com/a10gymbackupjotape-lang/a10-fitness-elite.git
   ```

2. **Instalar Dependências**:
   ```bash
   npm install
   ```

3. **Configurar Ambiente**:
   - Renomeie `.env.example` para `.env.local`.
   - Adicione suas credenciais do **Supabase** (URL e Anon Key).

4. **Modo Dev**:
   ```bash
   npm run dev
   ```
